# business-case

A three-phase business case system (English & Spanish). Run the phases in sequence — each consumes the previous one's output — or standalone:

| Phase | Skill | Output |
|---|---|---|
| 1 · Research | `business-case-research` | Consulting-grade market audit: sizing (TAM/SAM/SOM), competition, segmentation, Porter, trends — findings without a recommendation |
| 2 · Strategy | `business-case-strategy` | The decision layer: opportunity thesis, where-to-play / how-to-win, business model, pricing direction, **Go / No-Go / Pivot** with a scorecard |
| 3 · Activation | `business-case-activation` | Execution: unit economics, illustrative 12-month P&L direction, GTM plan, 30/60/90 roadmap, KPIs, kill-criteria |

**Install (Claude Code):** `/plugin install business-case@fk-toolkit` · after adding the marketplace: `/plugin marketplace add Fernando-Kalid/toolkit`

Then ask, e.g. *"audit this idea: …"* (phase 1), *"should I pursue it?"* (phase 2), *"build the plan"* (phase 3).
