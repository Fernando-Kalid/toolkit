# comms-studio

Marketing & communication suite — from brand strategy to landed campaign to on-page words:

| Skill | What it does |
|---|---|
| `comms-media-strategy` | Nine-module Communication & Media Strategy builder (ontology, semiotics, cultural analysis, behavioral science, commercial economics) — executive-ready |
| `campaign-development` | Compiles a strategy into one time-bound campaign: brief, insight, idea platform, seven-test quality gate, channel ecosystem, pre-registered measurement |
| `consent-based-ux-copywriting` | UX & conversion copy through a consent-based framework — strict anti-dark-pattern guardrails, plus copy audits |

**Install (Claude Code):** `/plugin install comms-studio@fk-toolkit` · after adding the marketplace: `/plugin marketplace add Fernando-Kalid/toolkit`

Typical flow: build the strategy → *"turn this into a campaign"* → *"write the landing page copy"*.
