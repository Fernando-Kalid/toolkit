# fk-decks

Branded presentation generators. Each skill produces on-brand PowerPoint `.pptx` decks (import cleanly into Google Slides) applying its brand's complete design system — surfaces, typography, accent rules, logos, and an executive component library (KPIs, charts, tables, timelines, dashboards).

| Skill | Brand |
|---|---|
| `fk-deck` | FERNANDO KALID — dark editorial personal brand |
| `ophira-deck` | OPHIRA — Wax Paper & Future Dusk, Fraunces + Inter |
| `success-planner-deck` | THE SUCCESS PLANNER / Winner's Circle — warm bone & onyx |

**Install (Claude Code):** `/plugin install fk-decks@fk-toolkit` · after adding the marketplace: `/plugin marketplace add Fernando-Kalid/toolkit`

Then just ask, e.g. *"haz una presentación con la marca OPHIRA sobre los resultados de Q3"* — the right skill triggers on its own.
